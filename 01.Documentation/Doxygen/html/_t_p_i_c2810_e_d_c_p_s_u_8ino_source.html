1/* Exercises the TPIC2810 in the EDCPSU v1 HW. No display, no rotary encoder, neither switches
2 * It uses a long array in program memory (PROGMEM) to store the array of values
3 * that drives the LDO through the 3 to 10 V span.
4 * There is a continuous ramping of values from 3 to 10V that takes about 1s to complete.
5 * Current consumption including Arduino board is from 100 to 115mA.
6 *
